1. **사용자 경험 최적화**: Core Web Vitals 기준 90점 이상 달성

4. **비용 최적화**: 인프라 비용 30% 절감하면서 성능 향상

  return function PerformanceWrapper(props: any) {
    useEffect(() => {
      performance.mark('component-start');
      return () => {
        performance.mark('component-end');
